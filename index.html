Ch.1 色のユニバーサルデザインとは
Q1 ユニバーサルデザイン（UD）とバリアフリーの違いは？ ▾
答え
UDは「最初からすべての人が使いやすい設計」、バリアフリーは「障壁を後から取り除く対処」という点で異なる。
  • バリアフリー：特定の障害に対し、後から障壁を除去する考え方
  • UD：年齢・性別・国籍・障害の有無にかかわらず、最初から誰もが使えるよう設計
  • 色のUD（CUD）：色覚の多様性に配慮し、誰もが見やすい色使いを実現すること
  • UC級では「バリアフリー」ではなく「UD（多様性の尊重）」の視点が重視される

Q2 色のUDの「3つのポイント」とは？ ▾
答え
①色を適切に選ぶ、②色以外の情報も併用する、③きちんと伝える（確認する）の3点。
  • ①色を選ぶ：混同しやすい色の組み合わせを避け、明度差・彩度差を活用する
  • ②色以外の情報を使う：形・模様・文字・アイコンなどを色と組み合わせる
  • ③きちんと伝える：シミュレーションや当事者確認でデザインを検証する

Ch.2 色が見えるしくみ
Q3 錐体（すいたい）と杆体（かんたい）の違いは？ ▾
答え
錐体は明所で色を識別する視細胞、杆体は暗所で明暗を識別する視細胞。色の区別は錐体だけが行う。
  • 錐体：L（長波長・赤）、M（中波長・緑）、S（短波長・青）の3種類
  • 杆体：1種類のみ。暗所で機能するが、色の判別はできない
  • 中心窩（ちゅうしんか）付近に錐体が集中している
  • L錐体・M錐体のどちらかに異常があると、赤～緑の識別が困難になる
  • 試験頻出：L/M/S錐体がどの色に対応するかをセットで覚える

Q4 可視光線の波長の範囲と色の並びは？ ▾
答え
可視光線は約380〜780nmで、波長が長い順に赤→橙→黄→緑→青→紫と並ぶ。
  • 赤：約780nm（最も長い）
  • 紫：約380nm（最も短い）
  • L錐体は長波長（赤）、M錐体は中波長（緑）、S錐体は短波長（青）に対応
  • 試験では「波長の長い順」と「L/M/S錐体の対応色」がよく問われる

Ch.3 色の表し方（色彩基礎）
Q5 色の三属性とは何か？ ▾
答え
色相・明度・彩度の3つ。これらを調整することで、混同しやすい色を見分けやすくできる。
  • 色相：赤・青・黄などの色みの違い（色の方向）
  • 明度：色の明るさ。高いほど白に近く、低いほど黒に近い
  • 彩度：色の鮮やかさ。高いほど純色に近い
  • UC級では「明度差を大きくする」ことが最も基本的な改善手法として頻出

Ch.4 色覚の多様性
Q6 色覚の型（C型・P型・D型・T型）の特徴は？ ▾
答え
C型が一般色覚。P型（1型）はL錐体、D型（2型）はM錐体に異常があり、赤と緑を見分けにくい。T型（3型）はS錐体の異常で青と黄の識別が困難。
  • C型（3色覚）：一般的な色覚。L・M・S錐体すべて正常
  • P型（1型色覚）：L錐体の異常。赤み～緑の識別が困難
  • D型（2型色覚）：M錐体の異常。赤～緑の識別が困難（日本人男性に最も多い）
  • T型（3型色覚）：S錐体の異常。青～黄の識別が困難。非常に稀
  • 日本人男性の約20人に1人（約5%）、女性の約500人に1人（約0.2%）がP型またはD型

Q7 先天性色覚異常と後天性色覚異常の違いは？ ▾
答え
先天性は遺伝的要因で生まれつき、後天性は疾病・加齢・薬の副作用などで生じる。
  • 先天性：遺伝子異常。男性に多い（X染色体連鎖）。P型・D型がほとんど
  • 後天性：眼疾患（緑内障・糖尿病網膜症など）、加齢、薬の副作用で発症
  • 後天性はT型（青黄型）が多い傾向がある
  • 加齢による水晶体の黄変もT型に近い色の見え方の変化をもたらす

Ch.5 高齢者の見え方
Q8 加齢が色の見え方に与える主な影響は？ ▾
答え
水晶体の黄変・白内障・コントラスト感度の低下により、特に青色系の見え方が変化し、色の識別力が落ちる。
  • 水晶体の黄変：黄色がかって見えるようになり、青〜紫が識別しにくくなる
  • 白内障：水晶体の混濁により全体が白くかすんで見える
  • コントラスト感度の低下：明暗の差が小さいと区別しにくくなる
  • 高齢者が見分けにくい例：黒と紺、白地に黄色の文字
  • 加齢は誰にでも起こるため、高齢社会においてUDの重要性は増す

Ch.6 見分けにくい色と配色の改善
Q9 P型・D型が見分けにくい色の組み合わせは？（混同色） ▾
答え
赤と緑、ピンクと水色、赤と茶、オレンジと黄緑などが代表的な混同色。
  • 赤と緑：最も代表的な混同色。特にD型（2型）に多い
  • ピンクと水色：試験公式例題にも登場（→紫みのピンクにすると区別しやすくなる）
  • 赤と茶、オレンジと黄緑も混同しやすい
  • 赤が暗く・くすんだ色に見えることがある
  • 日常の例：肉の焼き加減がわからない／充電ランプの色変化に気づかない

Q10 配色を識別しやすくする主な改善方法は？ ▾
答え
明度差を大きくすることが最も基本的な手法。加えて、色以外の情報（形・文字・模様）を組み合わせる。
  • 明度差を大きくする：最も効果的。低彩度の色同士の組み合わせは避ける
  • アウトライン（縁取り）を加える：背景と図の境界線をつけることで識別性が上がる
  • 文字・数字を加える：グラフの凡例に色だけを使わず、ラベルや数値も付記する
  • 模様・形を使う：折れ線グラフのマーカー形状、地図の塗り分けパターンなど
  • 色名を書き添える：マーカーペンなどに色名を記載することで伝達が確実に
  • セパレーション：色と色の間に白・黒・グレーを挟んで区別しやすくする

Ch.7 色のUDの進め方
Q12 色のUDを進める際のステップは？ ▾
答え
①多くの人が区別しやすい色を使う → ②色以外の手がかりを加える → ③有効性をチェックする → ④必要に応じ修正→完成。
  • ①色を選ぶ：明度差を確保、混同しやすい色系統の組み合わせを避ける
  • ②色以外の情報を加える：文字・形・模様・アンダーラインなど
  • ③有効性チェック：シミュレーションソフト、画像の白黒化、当事者に確認
  • 主なシミュレーションツール：Chromatic Vision Simulator、Adobe製品の機能など
  • ④修正・完成：問題点があれば修正し、完成へ

Q15 「混同色線」とは何か？（記述・穴埋め頻出） ▾
答え
色度図上に、色覚特性ごとに区別しにくい色の組み合わせを表示し、それらを直線で結んだもの。
  • P型・D型が混同しやすい色は、色度図上でほぼ一直線に並ぶ
  • その直線を「混同色線（こんどうしょくせん）」と呼ぶ
  • 選択肢に「ガンマ線」「エックス線」「無彩色軸」「混同色線」が並ぶ形で出題された実績あり
  • 無彩色軸はマンセル表色系の色立体の中心軸（黒〜白）のこと。混同色線とは別物

Q16 「1色覚」とは何か？P型・D型との違いは？ ▾
答え
1色覚は錐体が1種類しか働かない（または杆体のみ）色覚で、色みの区別がまったくできない。極めてまれ。
  • 1色覚（全色盲）：明るさは認識できるが、色を区別することはできない
  • 割合：日本人男性の0.001%と極めてまれ
  • P型（1型）・D型（2型）は「色みの区別が困難」だが、1色覚ではない
  • 試験では「色みの区別がまったくできない＝1色覚」という組み合わせで穴埋め出題あり

Q17 「ロービジョン」の定義は？ ▾
答え
病気やケガにより、眼鏡・コンタクトを使ってもよく見えない視力の低下や視野の欠損があり、生活に支障をきたす状態。
  • 原因：緑内障・糖尿病網膜症・遺伝性の網膜色素変性症・加齢黄斑変性症など
  • 色覚異常（P型・D型）とは異なる概念。視力・視野の問題
  • ロービジョンの人は青と黒、緑のピクトグラムが見分けにくい場合がある
  • 2018年に非常時避難ピクトグラムの緑色が黄みに変更されたのはこの配慮のため

Q18 「ヒュー・アンド・トーンシステム」とは？PCCSとの関係は？ ▾
答え
色を「色相（Hue）」と「トーン（Tone）」の2軸で分類するシステム。PCCSはこれを採用した日本の配色体系。
  • PCCS（日本色研配色体系）は1964年に日本色彩研究所が開発
  • 色相：24色相で構成。番号と記号で表す（例：2:R＝赤）
  • トーン：明度と彩度を組み合わせた概念。12種類（有彩色10＋無彩色系2）
  • 有彩色の表記例：v2（ビビッドの2番色相）のように「トーン記号＋色相番号」で表す
  • マンセル表色系とは異なり、トーンはPCCS独自の概念

Q19 「色立体」とは何か？ ▾
答え
色相・明度・彩度の三属性の関係を三次元的に表したもの。
  • 中心の縦軸：無彩色軸（黒から白）
  • 外側に向かうほど彩度が高くなる
  • 色相ごとに純色の明度が異なるため、立体の形は色相によって非対称
  • マンセル表色系にも色立体の概念あり（マンセルの木、Munsell Tree）

Q20 色の見え方が「反対色への分類」されるとはどういうことか？ ▾
答え
眼から入ったL・M・S錐体の情報が脳内で「赤または緑」「青または黄」の2対の反対色に変換・分類されること。
  • 色を感じる3段階：①錐体で光を受容 → ②反対色（赤/緑・青/黄）に分類 → ③色相・明度・彩度として分析
  • P型・D型はこの②の段階で「赤/緑」軸の信号が弱い・欠如している
  • T型は「青/黄」軸に問題がある
  • 試験では色が見えるプロセスの「順番」を問う穴埋め問題が出ることがある

深掘り3 具体的な事例・場面から考える系
Q25 電光掲示板の赤文字が見えにくい人がいる理由と改善策は？ ▾
答え
P型（1型色覚）は赤が暗く見えるため、黒い電光掲示板上の赤文字が背景と同化して見えにくい。オレンジ寄りに変更・白くすることで改善できる。
  • P型（1型）のみの特性：赤が暗く見える（D型よりも顕著）
  • 黒背景＋赤文字→P型には「黒＋黒（暗い色）」のように見える
  • 改善策：赤→オレンジ寄りに変更、または白文字に変更
  • カレンダーの日曜日表記も「赤→オレンジ寄り」への変更が進んでいる
  • 試験では「1型色覚に特有の見え方」として出題されやすい

Q27 折れ線グラフを色だけで区別している資料の、UCデザイン的な改善策は？ ▾
答え

  • マーカー形状を変える：●・▲・■・◆など異なる形状を使用

  • 明度差をつける：濃淡の差が大きい色の組み合わせにする
  • 色だけに頼らないことが最大の原則

Q28 高齢者が「黒と紺が同じ色に見える」のはなぜか？ ▾
答え
加齢による水晶体の黄変とコントラスト感度の低下により、暗い色同士の明度差が識別しにくくなるため。
  • 水晶体が黄変→青〜紫の波長が吸収されやすくなる
  • 黒と紺は本来明度が近いうえ、紺の青みが水晶体黄変でさらに識別しにくくなる
  • コントラスト感度の低下も重なり、暗色同士の区別がより困難に
  • 同様に見えにくい例：白地に黄色の文字、黒と茶、紺と緑など

Q29 「焦げ茶と深緑」が見分けにくい理由は？ ▾
答え
焦げ茶は赤を暗く・くすませたもの、深緑は緑を暗く・くすませたもの。赤と緑は混同色線上にあるため、明度・彩度を下げても同様に区別しにくい。
  • 赤と緑はP型・D型の混同色線上にある→同様に焦げ茶と深緑も区別しにくい
  • 「赤と緑が見分けにくい」だけでなく、その低彩度・低明度バリエーションも同様
  • 試験出題実績あり（2018年冬期 公式演習問題の選択肢に登場）
  • ピンクとグレーも同様の理由で区別しにくい（赤みのピンク＋無彩色の灰）

Q31 「色覚異常」という表現は試験で使っていいか？正しい表現は？ ▾
答え
試験・公式テキストでは「色覚の多様性」「色覚特性」「色覚タイプ」が使われる。「色覚異常」「色盲」は避けるべき古い表現。
  • ✅ 推奨表現：色覚の多様性 / 色覚特性 / 色覚タイプ / 1型・2型色覚
  • ❌ 避ける表現：色覚異常 / 色盲 / 色弱（ただし医学的文脈では使われる場合もある）
  • UC級の理念は「多様性の尊重」であり、否定的なニュアンスの用語は使わない
  • 記述式で「色覚異常」と書いても×にはならない可能性があるが、公式テキストの用語を使うのが無難
